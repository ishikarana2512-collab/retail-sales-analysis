# 🛒 Retail Sales Data Analysis & Dashboard

## 📌 Business Problem
Retail businesses need insights into sales performance, product demand, and regional profitability.

## 🎯 Objectives
- Analyze sales by category
- Identify profitable regions
- Track monthly trends
- Build Power BI dashboard

## 🛠 Tools Used
- Python (Pandas)
- Google Colab
- Power BI
- Excel

## 📊 Key Analysis
- Sales by Category
- Profit by Region
- Top 10 Products
- Monthly Sales Trends

## 📈 Dashboard
Includes KPI cards, filters, and interactive charts.

## 🔍 Key Insights
- High-performing categories drive major revenue
- Few products generate most sales
- Sales vary seasonally

## 💡 Business Recommendations
- Focus on top products
- Improve low-performing regions
- Use trends for forecasting
